Frequently Asked Questions

What is document object model (DOM)?

Dom is like a conainer who holds all the elements in the document.

How do I select an element from the DOM?

There is several methods to get a element. - getElementById(), getElementsByClassName(), querySelector(), getElementsByTagName() and querySelectorAll()

What is event delegation in the context of DOM, and why is it useful?

Event delegation is a technique involving adding event listeners to a parent element instead of adding them to the descendant elements. The listener will fire whenever the event is triggered on the descendant elements due to event bubbling up the DOM. And this is useful - because it allows I to only add a single event listener to handle potentially thousands of elements, which can be more efficient than adding individual event listeners to each of those elements.

How do I manipulate and elements attributes and styles using the DOM?

For attributes: I can use the getAttribute(), setAttribute(), and removeAttribute() methods.
For styles: I can use the style property of the element, which exposes an object with the element's inline styles. I can set individual styles using the style object's properties, or I can add or remove entire classes using the classList property.
